# m-home-archive

## 项目简介

m-home-archive 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源整理与历史版本留存使用。每个 HTML 页面均独立存放，便于直接访问、复制或嵌入其他环境。

## 目录说明

```
m-home-archive/
├── pages/           # 存放所有归档的 HTML 页面文件
├── assets/          # 公共资源文件（样式、脚本、图片等）
├── index.html       # 仓库入口页面（可选）
└── README.md        # 本文件
```

- `pages/`：按页面功能或主题分类，每个页面为一个独立的 `.html` 文件。
- `assets/`：存放页面引用的通用资源，避免重复加载。
- `index.html`：可选的汇总页面，用于快速跳转至各归档页面。

## 页面归档说明

本仓库中的 HTML 页面均为静态文件，不依赖后端服务。归档原则如下：

- 每个页面保持独立，不强制引用外部资源，但可引用 `assets/` 中的文件。
- 页面命名遵循 `[功能描述]-[版本或日期].html` 格式，便于识别版本。
- 归档内容不做功能保证，仅作为资料留存。

## 维护说明

- 欢迎提交 Pull Request 添加或更新页面，请确保文件符合上述目录结构。
- 若需删除或修改归档页面，请说明原因，维护者会进行审核。
- 本仓库不跟踪页面实际使用情况，仅关注文件本身的完整与可访问性。
